EDITOR INSTRUCTION

You can directly open White Dwarf Editor below without hosting any server. But to open a world file, you need to download the world files from the link below and open it in the editor.

1. Download world.json.
2. Open the downloaded file in the editor.
3. Right click to rotate views. Use scroll wheel to zoom in and out. Use middle click to move camera.
4. Change the mode on the top right corner from "move" to "view", now you can use left click to move the camera.

Mesh Subdivision Usage:
1. Create a new entity.
2. Add TransformData3D, IcosphereMeshGeneratorData, and MeshRenderData3D to the entity.
3. Change the flatNormal field to true for flattened normals.
4. Change the subdivision field to the number of subdivisions.
5. Change the radius field to the radius of the sphere.
6. You may not change the subdivision number too large. Set to 3 or 4 is recommended.
